1. Purpose
The governance layer defines how the substrate is controlled, how changes propagate, and how stability is enforced across all Prime‑era repos. It ensures that the substrate remains deterministic, reversible, and operator‑visible.

2. Governance Principles
The substrate governance model is built on four principles:

Explicit declaration — nothing enters the substrate implicitly

Deterministic structure — no ambiguous or emergent behavior

Operator primacy — the operator defines truth

Razor stabilization — contradictions are resolved through the Xi/Razor model

These principles form the substrate’s constitutional layer.

3. Decision Stability Rules (DSR)
The substrate enforces the DSR framework:

DSR‑1 — declaration, scope binding, Razor stabilization

DSR‑2 — lineage preservation

DSR‑3 — propagation boundaries

Each rule governs a different phase of substrate evolution.

4. Change Lifecycle
All changes to the substrate follow the Xi/Razor lifecycle:

Declared — operator intent is stated

Stabilized — contradictions resolved, scope bound

Propagated — change becomes part of the substrate

No change may skip a phase.
No change may be silently merged.

5. Naming Governance
The substrate enforces naming rules to maintain clarity and lineage:

directories use numeric prefixes for ordering

files use declarative names

no ambiguous or overloaded terms

naming changes require Razor stabilization

To define the naming rules:
Open naming conventions

6. Structural Governance
The substrate governs:

directory hierarchy

file placement

allowed surfaces

cross‑repo inheritance

No repo may introduce new top‑level surfaces without substrate approval.

To expand this:
Open repo standards

7. Enforcement Model
Governance is enforced through:

structural invariants

lineage checks

Razor contradiction resolution

operator review

Automation may assist, but authority remains with the operator.

8. Out of Scope
Governance does not define:

operator doctrine

runtime behavior

scanner logic

vessel logic

TUI surfaces

These belong to higher layers.
